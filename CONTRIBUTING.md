# Contributing to KRALN

Thank you for your interest in contributing!

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feat/your-feature`
3. Make your changes
4. Commit using conventional commits: `feat:`, `fix:`, `docs:`, `chore:`
5. Push and open a Pull Request

## Local setup

```bash
# Option 1 - VSCode Live Server (recommended)
# Right-click index.html -> Open with Live Server

# Option 2 - Python
python -m http.server 8080

# Option 3 - Node
npx serve .
```

## Guidelines

- Keep commits small and focused
- Test your changes in the browser before submitting
- Follow the existing code style
- Document any new features or mechanics

## Reporting bugs

Open an issue with:
- Description of the bug
- Steps to reproduce
- Expected vs actual behavior
