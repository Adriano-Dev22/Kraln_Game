## 📖 About
KRALN is a 2D side-scrolling action-adventure game with horror elements, developed entirely with HTML5 Canvas and pure JavaScript.

Descend into cursed biomes, collect powerful staves, cast devastating spells, and fight your way through increasingly horrifying levels.

## ✨ Features
- 🌐 **Bilingual** — English & Portuguese
- 🔮 **6 Magic Staves** — Fire, Frost, Lightning, Shadow, Poison, Bone
- 👾 **6 Enemy Types** — Ghosts, Skeletons, Demons, Bats, Wraiths, Zombies
- 💀 **Boss Fights** — Multi-phase bosses with rage modes
- 🗺️ **5 Levels** — Graveyard → Crypt → Abyss → Lich Keep → Inferno
- ⚡ **Status Effects** — Burn, Freeze, Poison, Lifesteal, Chain Lightning
- 🎯 **Varied Objectives** — Kill All, Boss Hunt, Shard Collection, Timed Escape
- ✨ **Particle Effects** — Trails, explosions, ambient fog
- 🛡️ **Combat** — Dash, Shield, Parry, Double Jump

## 🎮 Controls
| Action | Key |
|--------|-----|
| Move | A / D or Arrow Keys |
| Jump (double jump) | W / Space / ↑ |
| Crouch | S / ↓ |
| Cast Spell | X |
| Shield / Parry | C |
| Switch Staff | 1 / 2 / 3 / 4 |
| Dash | Shift |
| Interact | E |
| Pause | P / Esc |

## 🛠️ Local Development
```bash
# Option 1 — VSCode Live Server (recommended)
# Install "Live Server" extension → right-click index.html → Open with Live Server

# Option 2 — Python
python -m http.server 8080
# Open http://localhost:8080

# Option 3 — Node
npx serve .
```
